# Generative AI Practice Lab 🚀

This repository contains my **practice implementations and experiments in Generative AI**, including **RAG pipelines, LLM integrations, embeddings, vector databases, and AI applications**.

Most of the implementations are **practice codes inspired from Kaggle notebooks, research blogs, and official documentation**, which I study and reproduce to understand the concepts deeply.

The goal of this repository is to **build a strong practical understanding of modern Generative AI systems used in production**.

---

## 📚 What This Repository Contains

This repo includes hands-on practice code related to:

- Retrieval Augmented Generation (RAG)
- Large Language Models (LLMs)
- Vector Databases
- Embeddings
- LangChain pipelines
- Local LLM inference
- Document Question Answering
- Multi-modal AI pipelines

These implementations help understand how **real-world AI systems retrieve information and generate contextual answers.**
